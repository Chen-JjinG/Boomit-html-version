# 爆破超人 (Boomman) - HTML5 版

一个基于 HTML5 Canvas 和原生 JavaScript 开发的经典“泡泡堂/炸弹人”风格网页游戏。本项目具有智能 AI、多种武器系统以及丰富的游戏模式。

## 🎮 游戏特性

- **多角色颜色区分**：支持蓝色、红色、绿色、黄色四种固定颜色，方便玩家辨识。
- **丰富的武器系统**：
  - **经典炸弹**：延时爆炸，可破坏软墙。
  - **地雷**：隐形陷阱，踩中即爆。
  - **火箭筒**：直线远程攻击，即时碰撞引爆。
- **智能 AI 系统**：具备危险规避（逃离炸弹范围）、自动寻路和目标追踪能力的 Smart AI。
- **多种游戏模式**：
  - **单人游玩**：挑战多名 AI 敌人。
  - **双人对抗**：本地 P1 (WASD) vs P2 (方向键) 对战。
  - **AI 互博**：专门设计的观察模式，观看四个不同颜色的 AI 自动战斗。
  - **测试模式**：用于功能调试的特殊模式。

## 🚀 快速开始

1. 克隆本项目到本地：
   ```bash
   git clone https://github.com/Chen-JjinG/Boomit-html-version.git
   ```
2. 直接使用浏览器打开 `index.html` 即可开始游戏。

## ⌨️ 操作说明

### P1 (玩家 1)
- **移动**：`W`, `A`, `S`, `D`
- **动作 (放炸弹/发射火箭)**：`空格键`

### P2 (玩家 2)
- **移动**：`↑`, `↓`, `←`, `→`
- **动作 (放炸弹/发射火箭)**：`数字键 0` (或 `Insert`)

## 🛠️ 技术实现

- **核心框架**：原生 JavaScript (ES6+ Class)
- **渲染引擎**：DOM 操作与 CSS3 动画
- **AI 算法**：基于曼哈顿距离的危险地图分析与 A* 寻路优化
- **样式设计**：响应式 CSS 布局
